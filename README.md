
# IVM-VPS

Đây là script hỗ trợ cài đặt, thiết lập ứng dụng trên hệ điều hành Ubuntu để chạy ứng dụng quản lý invoice ( [ivm-app](https://github.com/sokhatvt/ivm-app) )

## Một số phần mềm cần thiết để vận hành hệ thống
- `NodeJS` phần mềm server side biên dịch ngôn ngữ javascript
- `MariadDB` quản lý cơ sở dữ liệu
## Một số phần mềm phụ trợ
- `NVM` quản lý cài đặt node version
- `PM2` đây là phần mềm quản lý ứng dụng nodejs (start, kill, hay restrart ứng dụng mỗi khi bị lỗi)
- `Nginx` là một ứng dụng tạo web server mạnh mẽ mã nguồn mở
- `SSL` tạo chứng chỉ cho web
